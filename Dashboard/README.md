## Power BI Sales Analytics Dashboard

This folder contains the Power BI (.pbix) file for the end-to-end sales analytics project.

### What this dashboard covers:
- Total Sales, Orders, and Customers KPIs
- Year-over-Year (YoY) and Last Year (LY) analysis
- Category and segment-level performance
- Time intelligence using a proper Date table
- Interactive slicers for business analysis

### Tools Used:
- Power BI
- Azure Fabric
- Spark SQL
- Microsoft Excel
